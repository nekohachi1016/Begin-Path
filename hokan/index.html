💻 自分のパソコンで遊んでみよう

これらのミニアプリは、ブラウザさえあれば誰でも動かせます。
HTMLやJavaScriptの経験がなくても大丈夫です！


Windowsの場合

  1. 「メモ帳」を開きます。
  2. HTML用ファイルを作ります：
    • 新規ファイルにHTMLコードをコピー＆ペースト
    • ファイル名を index.html にして保存
  3. JavaScript用ファイルを作ります：
    • 新規ファイルにJSコードをコピー＆ペースト
    • ファイル名を script.js にして保存
    • HTMLファイルと同じフォルダに置くこと
  4. HTMLファイルをダブルクリックすると、ブラウザでアプリが動きます！

Macの場合

  1. 「テキストエディット」を開きます。
  2. HTML用ファイルを作ります：
    • 新規書類にHTMLコードをコピー＆ペースト
    • フォーマットは 標準テキスト に変更
    • ファイル名を index.html にして保存
  3. JavaScript用ファイルを作ります：
    • 新規書類にJSコードをコピー＆ペースト
    • フォーマットは標準テキスト、ファイル名は script.js
    • HTMLファイルと同じフォルダに置くこと
  4. HTMLファイルをダブルクリックしてブラウザで開くと、アプリが動きます！

もしうまく動かないときは、
「HTMLとJSファイルが同じフォルダにあるか」「コードを正しくコピーできたか」を確認してください。


📱 スマホでも

スマホのブラウザや簡単なJavaScript実行アプリでも試せます。
外出先でもサクッと動かして遊んでみましょう！

  • iPhoneなら、JSAnywhereなどのアプリで実行可能
  • Androidなら、JavaScript For Androidなどのアプリで実行可能

HTML・JSファイルをコピペして動かせる点はPCと同じです。
